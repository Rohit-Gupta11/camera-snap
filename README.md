# Project Title

## Tech Stack
**Electron js**

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/Rohit-Gupta11/camera-snap.git
```

Go to the project directory

```bash
  cd camera-snap
```

Install dependencies

```bash
  npm install
```

Compile react source code

```bash
  npm run watch
```

Start the app

```bash
  npm run start
```
